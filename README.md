## tools
golang开发实用工具项目

### 搜集工具集合包含
```text
    DateUtils       date.DateTime
    IdUtils         id.Id
    EncodeUtils     encryption.Encryption
    IDCardUtils     idcard.Idcard
    StrUtils        str.Str
    FileUtils       file.ZFile
    EncryptionUtils encryption.Encryption
    HttpUtils       net.Client
```

### 项目中如何使用
> go mod init xxx

> 在项目中应用包路径 github.com/shenyuanheli/tools

> go mod tidy
